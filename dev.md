# monorepo 中的依赖与开发

`apps/h5-mobile` 依赖 `packages/ui-sdk-demo`

- 在 ui-sdk-demo 执行：pnpm run watch
- 在 `apps/h5-mobile` 中引用包：`"@sass/ui-sdk-demo": "workspace:*"`
- 启动 `apps/h5-mobile`，改动 `ui-sdk-demo` 后，即可自动更新内容
